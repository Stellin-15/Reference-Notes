# L02: Finder & File Management

Finder is the Mac equivalent of Windows File Explorer.

## Common shortcuts

- `Cmd+N` — new Finder window
- `Cmd+Shift+N` — new folder
- `Cmd+I` — get info on a file (size, permissions, kind — like Properties)
- `Space` — Quick Look: preview a file without opening it (huge time-saver, works on almost any file type)
- `Cmd+Delete` — move selected file(s) to Trash

## Copy vs move

- Dragging a file to another folder **moves** it by default (same volume/drive).
- Hold **Option** while dragging to **copy** it instead.
- Dragging between different drives/volumes copies by default.

## Trash

- Files go to Trash first, not deleted immediately (same as Windows Recycle Bin).
- Empty it: right-click the Trash icon in the Dock → **Empty Trash**.

## Useful Finder tricks

- `Cmd+Shift+G` — jump to a specific path by typing it (like typing a path in the Explorer address bar)
- `Cmd+Shift+.` — toggle showing hidden files (dotfiles)
- Sidebar (left panel) — pin frequently used folders by dragging them in
- Tags — right-click a file → Tags, for color-coded organization
