# L01: Basic Navigation & Keyboard Shortcuts

## Windows → Mac equivalents

| Windows | Mac |
|---|---|
| Taskbar | Dock (bottom of screen) |
| Start menu | Launchpad (F4, or pinch thumb+fingers on trackpad) |
| File Explorer | Finder |
| Control Panel | System Settings |
| Task Manager | Activity Monitor (Cmd+Space → type it) |
| Alt+Tab | Cmd+Tab (switch apps) / Cmd+` (switch windows in same app) |
| Right-click | Two-finger tap on trackpad, or Ctrl+click |

**Spotlight Search** (`Cmd+Space`) is the fastest way to do almost anything — launch apps, do math, search files, convert units. Use it instead of hunting through folders or the Dock.

## Essential keyboard shortcuts

Mac uses **Cmd (⌘)** where Windows uses Ctrl for most things:

- `Cmd+C` / `Cmd+V` / `Cmd+X` — copy / paste / cut
- `Cmd+Z` / `Cmd+Shift+Z` — undo / redo
- `Cmd+S` — save
- `Cmd+W` — close the current window/tab (does **not** quit the app)
- `Cmd+Q` — actually quit the app (apps keep running after you close their window — a dot appears under the Dock icon)
- `Cmd+Tab` — switch between open apps
- `Cmd+Space` — Spotlight search
- `Cmd+,` — open preferences for the current app
- `Cmd+Shift+4` — screenshot a selected area (saves to Desktop)
- `Cmd+Shift+3` — screenshot entire screen
- `Cmd+Shift+5` — screenshot/screen-recording tool with options
- `Cmd+Option+Esc` — force quit an app (like Ctrl+Alt+Del)
- `Cmd+Delete` — move a file to Trash

## Window management

- Green button (top-left of a window) — hover for fullscreen/tile options
- `Cmd+M` — minimize window
- Mission Control (swipe up with 3-4 fingers, or F3) — see all open windows/desktops
- Recent macOS (Sonoma and later) supports dragging windows to screen edges to tile them, similar to Windows Snap
